========================================
 tideWise 프로그램: 아주 쉬운 이용 방법
========================================

안녕하세요! tideWise를 사용해 주셔서 감사합니다.
이 프로그램은 어려운 주식 투자를 자동으로 도와주는 친구입니다.
아래 설명만 순서대로 따라오시면, 누구든지 쉽게 사용하실 수 있습니다.


[ 1. tideWise는 무엇을 하는 프로그램인가요? (주요 기능) ]

tideWise는 여러분 대신 주식을 사고파는 똑똑한 로봇 친구입니다!

🤖 주요 기능:
• 자동 단타매매: 하루 동안 주식을 사고팔아서 수익을 만들어드립니다
• 실시간 감시: 24시간 주식시장을 지켜보고 좋은 기회를 놓치지 않아요
• 안전장치: 너무 많이 잃지 않도록 미리 정해진 규칙으로 보호해드립니다
• 매매 기록: 언제 얼마나 사고팔았는지 예쁜 차트와 보고서로 보여드려요
• 모의투자 지원: 진짜 돈 사용하기 전에 가짜 돈으로 연습할 수 있어요


[ 2. 어떻게 똑똑하게 사고 팔까요? (매매 로직) ]

tideWise는 마치 숙련된 트레이더처럼 똑똑하게 판단합니다!

📊 매매 판단 방식:
• 거래량 폭증 감지: 갑자기 많은 사람들이 특정 주식을 사면 "뭔가 좋은 소식이 있나?"하고 매수해요
• 이동평균선 돌파: 주식 가격이 평균선을 위로 뚫고 올라가면 "상승세다!"하고 매수해요
• 데이비드 폴 분석: 거래량과 주가 움직임을 함께 보고 진짜 상승인지 가짜 상승인지 구분해요
• VI(급등급락 정지) 대응: 주식이 갑자기 너무 많이 오르거나 떨어져서 거래가 멈추면 즉시 대응해요

쉽게 말하면, 많은 사람이 관심 갖는 주식을 빠르게 사서 수익이 나면 팔고, 
손해가 날 것 같으면 빨리 손절하는 방식이에요!


[ 3. 이것만은 꼭 지키면서 거래해요! (매매 원칙) ]

tideWise는 여러분의 돈을 보호하기 위한 철칙이 있어요!

🛡️ 안전 규칙들:
• 손절매: 3% 이상 손해나면 무조건 팔아서 큰 손실을 막아요
• 익절매: 7% 이상 수익나면 욕심내지 말고 팔아서 수익을 확보해요
• 투자 한도: 가진 돈의 7%씩만 한 종목에 투자해서 위험을 분산해요
• 최대 5개: 동시에 최대 5개 종목까지만 보유해서 관리를 쉽게 해요
• 장마감 30분전 정리: 오후 3시 이후엔 새로 사지 않고 가진 것만 정리해요
• 일일 손실 한도: 하루에 총 자산의 5% 이상 잃으면 그날은 매매를 중단해요

이 규칙들 덕분에 큰 손실 없이 안전하게 투자할 수 있어요!


[ 4. 시작 전, 반드시 설정해야 하는 것들! (필수 정보 입력) ]

프로그램을 사용하기 위해서는 한국투자증권의 'API 키'가 반드시 필요해요!

🔑 준비물 (한국투자증권에서 발급받아야 하는 4가지):
1) APP KEY (앱 열쇠) - 예: PSKmaksidf2k3idkf2k3asdkf2k
2) APP Secret KEY (비밀 앱 열쇠) - 예: ADFJasdkf23kjasdf92k3jasdkf2k3jasdfk23
3) 계좌번호 - 예: 12345678-01 (8자리숫자-01 형태)
4) 계좌 비밀번호 - 예: 1234 (4자리 숫자)

🌟 한국투자증권 API 발급받기 (처음 하시는 분 필수!):

▶️ 1단계: 한국투자증권 홈페이지 접속
   - https://www.koreainvestment.com 으로 이동
   - 로그인하세요 (계좌가 없으면 먼저 계좌개설 필요)

▶️ 2단계: API 서비스 신청
   - 홈페이지 상단 메뉴에서 "트레이딩/투자" → "OPEN API" 선택
   - "API 서비스 신청" 버튼 클릭
   - 본인인증 진행 (휴대폰 인증)

▶️ 3단계: 2개 종류의 API 발급받기
   📌 실전투자용 API (진짜 돈으로 거래):
      - "실전투자 API" 신청
      - 발급받은 정보: 실전 APP KEY, 실전 APP Secret KEY
      - 계좌번호: 본인의 실제 계좌번호-01
      - 계좌 비밀번호: 실제 거래 비밀번호 4자리
   
   📌 모의투자용 API (연습용 가상 돈):
      - "모의투자 API" 신청 (별도 신청 필요!)
      - 발급받은 정보: 모의 APP KEY, 모의 APP Secret KEY
      - 계좌번호: 모의투자용 가상 계좌번호 (보통 50000000-01 형태)
      - 계좌 비밀번호: 모의투자용 비밀번호 4자리

💡 꿀팁: 처음에는 반드시 모의투자 API부터 신청해서 연습하세요!

📝 tideWise에 API 정보 설정하기 (초상세 가이드):

🎯 1단계: tideWise 프로그램 실행
   - 프로그램 폴더에서 "run.py" 파일을 더블클릭
   - 검은 화면이 뜨면서 로딩... (30초 정도 기다려주세요)
   - 예쁜 메뉴가 나타나면 성공!

🎯 2단계: 설정 메뉴 진입
   - 화면에서 숫자 "3"을 입력하고 Enter 키 누르기
   - "Setup" 메뉴로 들어가짐

🎯 3단계: API 정보 입력 화면 열기
   - 숫자 "1"을 입력하고 Enter 키 누르기
   - "Register_Key (통합 연동정보 관리)" 선택됨
   - 자동으로 메모장(또는 편집기)이 열립니다!

🎯 4단계: 메모장에서 정보 수정하기 (매우 중요!)
   메모장에 이미 틀이 만들어져 있어요. 아래 부분을 찾아서 수정하세요:

   📍 실전투자 계좌 정보 섹션:
   ```
   ### 실전투자 계좌 정보
   
   APP KEY: [여기에 실전 APP KEY를 입력하세요]
   APP Secret KEY: [여기에 실전 Secret KEY를 입력하세요]
   계좌번호: [여기에 실제 계좌번호-01을 입력하세요]
   계좌 비밀번호: [여기에 실제 4자리 비밀번호를 입력하세요]
   ```

   📍 모의투자 계좌 정보 섹션:
   ```
   ### 모의투자 계좌 정보
   
   APP KEY: [여기에 모의 APP KEY를 입력하세요]
   APP Secret KEY: [여기에 모의 Secret KEY를 입력하세요]
   계좌번호: [여기에 모의투자 계좌번호를 입력하세요]
   계좌 비밀번호: [여기에 모의투자 4자리 비밀번호를 입력하세요]
   ```

   ⚠️ 주의사항:
   - 대괄호 [ ] 도 제거하고 실제 정보를 입력하세요
   - APP KEY는 보통 30자 정도의 긴 영문+숫자 조합이에요
   - Secret KEY는 보통 40자 정도의 매우 긴 영문+숫자 조합이에요
   - 계좌번호는 반드시 끝에 "-01"이 붙어요
   - 공백이나 특수문자가 들어가지 않도록 주의하세요!

🎯 5단계: 저장하고 닫기
   - Ctrl+S를 눌러서 저장하세요
   - 메모장을 닫으세요
   - 프로그램 화면으로 돌아가면 "Enter를 눌러주세요"라고 나올 거에요
   - Enter 키를 누르세요

🎯 6단계: 설정 완료 확인
   - "설정이 완료되었습니다!" 메시지가 나타나면 성공!
   - 만약 오류 메시지가 나타나면 다시 1단계부터 해주세요

🔐 추가 보안 설정 (선택사항):
프로그램이 "암호화를 설정하시겠습니까?"라고 물어보면:
- "Y" 입력: API 정보를 암호화해서 더 안전하게 보관 (추천!)
- "N" 입력: 그냥 일반 텍스트로 보관

🧪 텔레그램 알림 설정 (선택사항):
매매 결과를 핸드폰으로 받고 싶다면:
1) 텔레그램 앱에서 "@BotFather" 검색해서 대화
2) "/newbot" 명령어로 봇 생성
3) 받은 "Bot Token"을 메모장의 Bot Token 부분에 입력
4) 본인 텔레그램의 "Chat ID"를 Chat ID 부분에 입력

❌ 자주 하는 실수들:
- APP KEY에 공백 포함: "PSK maksi..." (X) → "PSKmaksi..." (O)  
- 계좌번호에서 -01 빼먹기: "12345678" (X) → "12345678-01" (O)
- 대괄호 그대로 두기: "[12345678-01]" (X) → "12345678-01" (O)
- 실전/모의 정보 섞어서 입력하기 (각각 따로 발급받아야 함!)

💯 설정 성공 확인 방법:
설정 후 프로그램 메인 메뉴에서 "1번 단타매매" → "2번 모의단타매매" 선택했을 때:
- "API 연결 확인 중..." 메시지가 나타남
- "연결 성공! 계좌정보를 불러왔습니다" 메시지가 나타나면 완벽!
- 오류가 나타나면 설정을 다시 확인해보세요


[ 5. 이제 함께 시작해봐요! (초간단 설정 방법) ]

이제 정말 쉬워요! 클릭 몇 번이면 끝이에요!

🚀 프로그램 첫 실행:
1단계: 프로그램 폴더에서 "run.py" 파일을 더블클릭해주세요
2단계: 검은 화면이 뜨면서 프로그램이 시작됩니다 (조금 기다려주세요)
3단계: 예쁜 메인 메뉴가 나타납니다!

📋 메인 메뉴 설명:
• 1번: 단타매매 - 하루 동안 자동으로 주식 사고팔기
• 2번: 단타매매 알고리즘변경 - 다른 매매 방식으로 바꾸기
• 3번: Setup - 각종 설정을 쉽게 바꾸기 (중요!)
• 0번: 종료 - 프로그램 끝내기

⚙️ Setup 메뉴로 기본 설정하기:
메인 메뉴에서 "3번 Setup"을 선택하면 설정 메뉴가 나타납니다:

• 1번: Register_Key (통합 연동정보 관리)
  → 한국투자증권 API 키와 계좌번호를 입력하는 곳이에요
  → 가장 중요한 설정이니 제일 먼저 해주세요!

• 2번: 지정테마 설정  
  → 어떤 테마의 주식을 매매할지 정하는 곳이에요
  → 예: 반도체, 바이오, 게임 등 관심 있는 분야 선택

• 3번: 지정종목 설정
  → 특별히 관심 있는 개별 종목들을 추가하는 곳이에요
  → 예: 삼성전자, LG에너지솔루션 등 직접 지정

• 4번: 장마감 체크 설정
  → 오후 2시 이후 매매를 계속할지 멈출지 정하는 곳이에요
  → ON: 2시 후 완전 중단, OFF: 5분마다 계속 확인

💡 처음 사용하시는 분들에게 추천 순서:
1) 메인 메뉴에서 "3번 Setup" → "1번 Register_Key" 먼저 설정
2) 설정이 끝나면 "1번 단타매매" 선택
3) "2번 모의단타매매" 선택 (연습용으로 시작!)
4) 프로그램이 자동으로 주식을 사고팔기 시작해요!
5) 익숙해지면 "1번 실전단타매매"로 진짜 돈 투자하기

📊 결과 확인:
매매가 끝나면 "report" 폴더에 예쁜 차트와 보고서가 자동으로 만들어져요!
- session.html 파일: 웹브라우저로 열면 예쁜 차트를 볼 수 있어요
- session.json 파일: 상세한 매매 기록이 저장되어 있어요

🎉 축하합니다! 이제 tideWise와 함께 스마트한 투자를 시작하세요!

🔧 추가 설정 가이드:

📈 기본 매매 설정 변경하기:
Setup 메뉴에서 다른 설정들도 본인 취향에 맞게 바꿀 수 있어요:

🎨 2번: 지정테마 설정
   - 반도체, 바이오, 게임 등 관심 있는 테마 선택
   - "반도체"를 선택하면 삼성전자, SK하이닉스 같은 반도체 주식만 거래
   - 여러 테마를 동시에 선택할 수도 있어요!

📋 3번: 지정종목 설정  
   - 특별히 관심 있는 개별 종목 직접 추가
   - 예: "005930" (삼성전자 코드) 입력하면 삼성전자만 거래
   - 최대 10개까지 종목 추가 가능해요

⏰ 4번: 장마감 체크 설정
   - ON: 오후 2시 이후 새로운 매수 중단 (안전)
   - OFF: 오후 3시 20분까지 계속 매매 (적극적)
   - 처음에는 ON 추천!

🛠️ 고급 설정 (고수용):
   - 손절매 비율: 기본 -3% (더 보수적으로 하려면 -2%)
   - 익절매 비율: 기본 +7% (욕심 부리지 말고 유지 추천)
   - 최대 보유 종목: 기본 5개 (초보자는 3개 추천)

⚠️ 처음 사용자 주의사항:
1. 반드시 모의투자로 최소 1주일 연습 후 실전 시작
2. 실전 시작할 때는 소액(10-50만원)부터 시작  
3. 프로그램을 너무 자주 끄고 켜지 마세요 (매매 기회 놓칠 수 있어요)
4. 시장이 너무 불안한 날(급락하는 날)에는 수동으로 중단하세요

💡 더 자세한 API 발급 방법은 위의 "🌟 한국투자증권 API 발급받기" 섹션을 참고하세요!

행복한 투자 되세요! 🍀
